# E-Commerce Data Analysis Dashboard ✨

## Deskripsi
Project ini bertujuan untuk menganalisis data e-commerce dan menyajikan insight dalam bentuk dashboard interaktif menggunakan Streamlit.

## Pertanyaan Bisnis
1. Kategori produk apa yang memberikan kontribusi terbesar terhadap penjualan?
2. Bagaimana tren jumlah pesanan dari waktu ke waktu?

## Setup Environment
1. Install dependencies:
   pip install -r requirements.txt

## Run Streamlit App
streamlit run dashboard.py

## Insight Utama
- Kategori dengan jumlah pembelian terbanyak tidak selalu memiliki total penjualan terbesar.
- Tren pesanan meningkat hingga akhir tahun 2017 dan cenderung stabil setelahnya.
